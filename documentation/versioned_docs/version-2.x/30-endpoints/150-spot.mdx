<ArtifactInstall repo="spot" />

## Client instantiation

```java
SpotClient client = new SpotClient(key, secret);
```

## Testnet client instantiation

```java
TestnetSpotClient client = new TestnetSpotClient(testnetKey, testnetSecret);
```
