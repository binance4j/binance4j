Fiat endpoints

  

Installation​

Apache Maven

pom.xml
<dependency>
	<groupId>{{groupId}}</groupId>
	<artifactId>{{artifactId}}</artifactId>
	<version>{{version}}</version>
</dependency>

Gradle Groovy DSL

build.gradle
implementation '{{groupId}}:{{artifactId}}:{{version}}'

Client instantiation​

FiatClient client = new FiatClient(key, secret);
