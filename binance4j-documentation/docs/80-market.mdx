## Features

### Test connectivity

```java
client.ping().sync();
```

### Get server time

<Examples payload="ServerTimeRespons" method="getServerTime" items={[{}]} />

### Get exchange info

<Examples
    payload="ExchangeInfo"
    method="getExchangeInfo"
    items={[
        { label: "All symbols" },
        {
            label: "One symbol",
            params: "new ExchangeInfoParams(symbol)",
        },
        {
            label: "Multiple symbols",
            params: "new ExchangeInfoParams(symbols[])",
        },
    ]}
/>

### Get a symbol's order book

<Examples
    payload="OrderBook"
    method="getOrderBook"
    items={[
        { label: "One symbol", params: "new OrderBookParams(symbol)" },
        {
            label: "With limit (Enum)",
            params: "new OrderBookParams(symbol,OrderBookLimit.LIMIT_10)",
        },
        {
            label: "With limit (String)",
            params: 'new OrderBookParams(symbol,"10")',
        },
    ]}
/>

:::caution
The String value must match one of the `OrderBookLimit` values or the request will fail
:::
